Markdown基础语法

Created2024-10-14|Updated2024-10-15|Tools
|Word Count:1.1k|Reading Time:3mins|Post Views:

Markdown 是一种轻量级的标记语言，能够通过简单的标记符号来编写格式化文本。它的设计初衷是让文本既可以轻松阅读，又能够直接转换成 HTML，非常适合编写博客、文档和 README 文件。

基础语法

1. 标题（Headers）

使用 # 符号来定义标题，# 的个数代表标题的层级（从 # 到 ######，依次为 1-6 级标题）。

2. 段落（Paragraphs）

  • Markdown 中段落之间需要空一行，直接输入文字即可。

渲染效果：

这是一个段落。

这是另一个段落。

  • 或者使用HTML语法

渲染效果：

这是另一种方法的一个段落。

3. 强调（Emphasis）

可以使用星号 * 或下划线 _ 来进行文字的强调。

  • 斜体：使用一个星号或下划线包裹文字。
  • 加粗：使用两个星号或下划线包裹文字。
  • 加粗并斜体：使用三个星号或下划线包裹文字。

渲染效果：

斜体
加粗
加粗并斜体

4. 列表（Lists）

有序列表和无序列表的创建也非常简单。

无序列表：

使用 *、+ 或 - 表示无序列表。

渲染效果：

  • 项目一
  • 项目二
  • 项目三

有序列表：

使用数字加句点 . 表示有序列表。

渲染效果：

  1. 第一项
  2. 第二项
  3. 第三项

5. 链接（Links）

可以通过方括号 [] 包含显示文本，后跟圆括号 () 包含链接地址的方式创建链接。

9. 表格（Tables）

使用管道符号 | 创建表格，使用短横线 - 来分隔表头和内容。

渲染效果：

名字 年龄
小明 18
小红 20

管道字符 |

  • 定义列的边界

  • 第一行通常是表头，第二行是用短横线 (-) 来定义表格的结构（并且同样使用管道符号来分隔列）。

  • 每个单元格的内容可以有不同的对齐方式。

    对齐方式
    你可以通过在短横线旁边添加 : 来定义表格中列的对齐方式：

    • :— 表示左对齐
    • —: 表示右对齐
    • :—: 表示居中对齐
