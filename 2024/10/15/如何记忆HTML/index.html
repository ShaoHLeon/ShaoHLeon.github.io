如何记忆HTML

Created2024-10-15|Updated2024-10-31|Web
|Word Count:1.7k|Reading Time:6mins|Post Views:

HTML标签记忆

标签 英文 功能
html hypertext markup language 超文本标记语言，定义HTML文档
body body 定义文档主题，可显示部分
h heading 标题
段落:
p paragraph 段落
br break 换行
hr horizontal rule 定义水平分割线，用于分隔内容
文本格式化标签:
b bold 将文本加粗
strong strong 强调文本（通常呈现为加粗）
em emphasis 强调文本（通常呈现为斜体）
i italic 将文本斜体
sub subscript 下标字
sup superscript 上标字
ins insert 插入字
del delete 删除字
“计算机输出” 标签:
code code 计算机输出的代码片段
kbd keyboard 键盘输入,键盘码
tt teletype 打字机文本
samp sample 计算机代码样本
var variable 计算机变量
pre preformatted 预格式文本
引文, 引用, 及标签定义
abbr abbreviation 定义缩写
address address 定义地址信息
bdo bidirectional override 定义文字方向
blockquote blockquote 定义长的引用
q quote 定义短的引用语
cite citation 定义引用、引证
dfn definition 定义一个定义项目
链接
a anchor 锚点，超链接
头部
head head 包含了所有的头部标签元素。
title title 定义网页的标题，显示在浏览器标签页上
style style 定义内部CSS样式，直接在HTML文档中编写样式
meta metadata 定义元数据，如网页描述、关键词、字符编码等
link link 定义外部资源链接，通常用于引入外部CSS或其他文件
script script 定义客户端脚本，如JavaScript代码
noscript no script 当浏览器不支持或禁用了JavaScript时显示的内容
base base 定义页面上所有相对链接的基础URL
样式标签
style style 定义内联样式，直接在HTML文档中编写样式
link link 定义外部资源链接，通常用于引入外部CSS或其他文件
图片标签:
img image 定义图像
map map 定义一个图像映射，用于定义点击区域，并执行不同的操作。
area area 定义图像映射中的区域，通常与 <map> 标签一起使用。
表格
table table 定义表格
tr table row 表示表格的一行
td table data 表示表格的数据单元格
th table header 表示表格的表头单元格
caption caption 定义表格的标题
colgroup column group 定义表格列的组
col column 定义用于表格列的属性
thead table head 定义表格的页眉部分
tbody table body 定义表格的主体部分
tfoot table footer 定义表格的页脚部分
列表
ul unordered list 定义无序列表
ol ordered list 定义有序列表
li list item 表示列表中的项目
dl definition list 自定义列表
dt definition term 定义自定义列表中的项目
dd definition description 定义自定义列表项的描述
区块
div division 定义文档中的一个区块
span span 定义行内区块
表单和输入
form form 定义一个表单，用于用户输入数据
label label 定义表单字段的标签，与 <input> 标签一起使用
select select 定义下拉列表，用于从多个选项中选择一个值
input input 定义用户输入的字段，如文本框、复选框、按钮等
option option 定义下拉列表中的一个选项
textarea text area 定义多行文本输入字段，通常用于长文本输入。
fieldset field set 定义表单字段集，通常用于将多个表单元素组合在一起，并添加样式。
legend legend 定义表单字段集的标题，通常与 <fieldset> 标签一起使用。
outgroup out group 定义一个输出字段，用于显示由脚本生成的内容。
button button 定义一个按钮，用于触发脚本或提交表单。
datalist data list 预先定义的输入控件选项列表，通常与 <input> 标签一起使用。
框架
iframe inline frame 定义内嵌框架，即在当前页面中显示其他页面的内容。
脚本
script script 定义客户端脚本，如JavaScript。
noscript no script 当浏览器不支持或禁用了JavaScript时显示的内容。
字符实体
  non-breaking space 插入一个空格，但不换行。
< less than 小于号。
> greater than 大于号。
& ampersand 插入一个 & 字符。
" quotation mark 插入一个双引号。
' apostrophe 插入一个单引号。

HTML属性记忆

属性：

  • 在元素中添加属性
  • 描述于开始标签
  • 以名字/值对形式出现
属性 英文 功能
链接属性
href hyperlink reference 指定超链接的目标地址
target target 定义链接的打开方式
rel relationship 定义链接与目标页面的关系，例如 nofollow, noopener, noreferrer。
nofollow nofollow 告诉搜索引擎不跟踪该链接，常用于外部链接。
noopener noopener 防止新页面访问 window.opener 属性，避免安全问题，通常与 target="_blank" 配合使用。
noreferrer noreferrer 不发送 referer header，防止目标页面知道你从哪里来。
download download 提示浏览器下载链接的目标，而不是导航到该目标。
title title 定义链接的额外信息，鼠标悬停时显示工具提示。
id id 用于链接锚点，通常用于跳转到页面中的特定位置。
hreflang hreflang 指定目标URL的语言。
type type 指定链接资源的MIME类型，例如 text/css。
class class 用于指定元素的类名，便于CSS样式的应用。
style style 直接在元素上定义CSS样式。
图像属性
src source 指定外部资源的来源，如图片、视频、脚本等
alt alternate 当图片无法显示时，提供的替代文本
width width 指定图像的宽度，单位为像素
height height 指定图像的高度，单位为像素
align align 定义图像的垂直对齐方式，如 left, right, center。
表格属性
border border 指定表格边框的宽度，单位为像素。
colspan column span 定义单元格跨越的列数
rowspan row span 定义单元格跨越的行数
cellspacing cell spacing 指定单元格之间的间距，单位为像素。
cellpadding cell padding 指定单元格内容与边框之间的间距，单位为像素。
表单属性
action action 指定表单提交的目标URL，默认为当前页面。
method method 指定表单提交的方式，如 get(default), post。
enctype encoding type 定义表单数据在发送到服务器时的编码类型
name name 定义表单的名称，用于提交数据时使用。
value value 定义表单字段的默认值。
checked checked 用于定义复选框或单选按钮的默认选中状态。
title title 为元素提供提示信息，通常用于鼠标悬停时显示
id identifier 为元素定义唯一的标识符，方便CSS、JavaScript引用
class class 为元素定义一个或多个类名，通常用于CSS样式控制
type type 指定元素的类型，如 <input> 标签的类型、脚本类型等
