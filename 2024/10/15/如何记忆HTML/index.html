HTML标签记忆

标签 英文 功能
html hypertext markup language 超文本标记语言，定义HTML文档
body body 定义文档主题，可显示部分
h heading 标题
p paragraph 段落
a anchor 锚点，超链接
br break 换行
div division 定义文档中的一个区块
span span 定义行内区块
input input 定义用户输入的字段，如文本框、复选框、按钮等
hr horizontal rule 定义水平分割线，用于分隔内容
b bold 将文本加粗
strong strong 强调文本（通常呈现为加粗）
em emphasis 强调文本（通常呈现为斜体）
i italic 将文本斜体
code code 定义计算机输出的代码片段
kbd keyboard 定义键盘输入
tt teletype 定义打字机文本
samp sample 定义计算机代码样本
var variable 定义计算机变量

HTML属性记忆

属性：

  • 在元素中添加属性
  • 描述于开始标签
  • 以名字/值对形式出现
属性 英文 功能
href hyperlink reference 指定超链接的目标地址
src source 指定外部资源的来源，如图片、视频、脚本等
alt alternate 当图片无法显示时，提供的替代文本
title title 为元素提供提示信息，通常用于鼠标悬停时显示
id identifier 为元素定义唯一的标识符，方便CSS、JavaScript引用
class class 为元素定义一个或多个类名，通常用于CSS样式控制
style style 在元素内直接定义CSS样式
type type 指定元素的类型，如 <input> 标签的类型、脚本类型等
